# Лаборатория потока

Кликабельная канбан-доска на русском языке с уникальной фишкой: **Ассистент фокуса**. Первый экран — рабочая доска: toolbar, компактный ассистент, строка выбранной карточки и канбан-колонки на всю ширину.

Доска решает частую боль канбана: карточки стареют, блокируются или теряются в перегруженных колонках, а обычная доска не говорит пользователю, что лучше закрыть первым.

## Запуск

```bash
npm run dev
```

Открыть `http://localhost:5173`.

## Проверка

```bash
npm run build
```

Команда запускает синтаксическую проверку JavaScript и тесты доменной логики доски.

## Рабочий флоу

1. Открыть пустую доску.
2. Нажать `Новая задача` и создать первую карточку.
3. Нажать `?` рядом с **Ассистентом фокуса**, чтобы посмотреть, почему он выбирает первую карточку.
4. Перемещать карточку кнопками `Вперед` / `Назад` или drag/drop между колонками.
5. Отмечать блокировку, разделять слишком крупную карточку или отправлять завершенную работу в архив.
6. Открыть `Архив` и вернуть карточку обратно на доску, если ее убрали по ошибке.
7. Смотреть на **Ассистент фокуса**: он пересчитывает рекомендации по текущему состоянию доски.

## Уникальная фишка

Ассистент оценивает незакрытые карточки по нескольким сигналам:

- сколько дней карточка стоит в текущей колонке;
- заблокирована ли она;
- перегружена ли текущая колонка по WIP;
- какой у карточки приоритет;
- близко ли срок;
- насколько карточка маленькая и быстро закрываемая.

После этого он дает короткую рекомендацию: `закрыть проверку`, `разделить или закрыть`, `разблокировать или откатить`.
