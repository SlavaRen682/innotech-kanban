# Дизайн канбан-доски

## Цель

Сделать кликабельную канбан-доску с одной защищаемой уникальной фишкой: Ассистент фокуса.

## Объем

Приложение — локальное статическое web-приложение. Оно стартует с пустой доски и поддерживает ручное создание карточек, drag/drop, выбор карточки, перемещение по workflow, блокировку, разделение, архив и сохранение в браузере.

## Уникальная фишка

Ассистент фокуса ранжирует незакрытые карточки по возрасту в текущей колонке, блокировке, WIP-перегрузу, приоритету, риску срока и размеру. Он показывает три главные карточки и рекомендует одно действие для каждой.

## UI-направление

Использовать приложенный high-end visual design skill как визуальное ограничение: премиальные double-bezel панели, тактильные pill-кнопки, кастомное cubic-bezier движение, без запрещенных шрифтов и с адаптацией ниже tablet width.

## Архитектура

Доменная логика находится в `src/kanban-core.js`, UI-поведение — в `src/app.js`. Для сохранения состояния используется `localStorage`, для проверки — встроенный Node test runner.

## Проверка

Запустить `npm run build`, затем smoke-test в браузере на `http://localhost:5173`.
