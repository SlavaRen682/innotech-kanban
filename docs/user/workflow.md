# Рабочий сценарий

Последовательность для проверки проекта:

1. Открыть доску и убедиться, что она стартует пустой.
2. Нажать `Новая задача`, заполнить название, описание, ответственного, выбрать приоритет и размер сегментами, указать срок и теги.
3. Нажать `?` рядом с `Ассистент фокуса` и проверить SLE-формулу: `Оценка = max(0, min(возраст / SLE, 1.4) * 45 + бонус SLE + стадия + приоритет + размер + срок + зависание + блокер + WIP)`.
4. Выбрать созданную карточку и проверить строку действий.
5. Нажать `Вперед`, `Заблокировать`, `Разделить` или `В архив`.
6. Нажать `Архив`, найти убранную карточку и нажать `Вернуть`.
7. Перетащить карточку между колонками и проверить WIP-метры.
8. Создать несколько карточек с разными сроками и приоритетами, чтобы увидеть рекомендации `Что закрыть первым`.
