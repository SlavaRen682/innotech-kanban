# Рабочий сценарий

Последовательность для проверки проекта:

1. Открыть доску и убедиться, что она стартует пустой.
2. Нажать `Новая задача`, заполнить название, описание, ответственного, выбрать приоритет и размер сегментами, указать срок и теги.
3. Выбрать созданную карточку и проверить строку действий.
4. Нажать `Вперед`, `Заблокировать`, `Разделить` или `В архив`.
5. Нажать `Архив`, найти убранную карточку и нажать `Вернуть`.
6. Перетащить карточку между колонками и проверить WIP-метры.
7. Создать несколько карточек с разными сроками и приоритетами, чтобы увидеть рекомендации `Что закрыть первым`.
