# Лаборатория потока

Кликабельная учебная демка канбан-доски с исследованной уникальной фишкой: **Ассистент фокуса**. Интерфейс сделан как чистая рабочая доска: без лендинговых hero-блоков и промо-секций.

Доска решает частую боль канбана: карточки стареют, блокируются или теряются в перегруженных колонках, а обычная доска не говорит пользователю, что лучше закрыть первым.

## Запуск

```bash
npm run dev
```

Открыть `http://localhost:5173`.

## Проверка

```bash
npm run build
```

Команда запускает синтаксическую проверку JavaScript и тесты доменной логики доски.

## Сценарий демо

1. Открыть доску и посмотреть на **Ассистент фокуса**.
2. Выбрать первую рекомендованную карточку.
3. Нажать `Вперед`, `Заблокировать`, `Разделить` или `В архив`.
4. Перетащить карточку между колонками и показать, как меняются WIP-метры и рекомендации.
5. Нажать `+1 день`, чтобы показать старение карточек в статусах.
6. Нажать `Сбросить демо`, чтобы вернуться к подготовленному сценарию.

## Уникальная фишка

Ассистент оценивает незакрытые карточки по нескольким сигналам:

- сколько дней карточка стоит в текущей колонке;
- заблокирована ли она;
- перегружена ли текущая колонка по WIP;
- какой у карточки приоритет;
- близко ли срок;
- насколько карточка маленькая и быстро закрываемая.

После этого он дает короткую рекомендацию: `закрыть проверку`, `разделить или закрыть`, `разблокировать или откатить`.
