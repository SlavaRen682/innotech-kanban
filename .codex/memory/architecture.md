# Архитектура

## Текущее состояние

Статическая браузерная демка канбан-доски с Ассистентом фокуса. Весь видимый продуктовый слой должен быть на русском языке.

## Модули

- `src/kanban-core.js`: чистая модель доски, демо-карточки, WIP-проверки, скоринг зависших карточек, focus ranking, действия над карточками и сводки.
- `src/app.js`: DOM-рендеринг, обработка событий, drag/drop, localStorage, состояние модалки и reveal-анимации.
- `src/styles.css`: премиальная визуальная система с double-bezel панелями, адаптивной доской, WIP-метрами, age states и кастомным motion.
- `tests/kanban-core.test.mjs`: behavior tests для core-модели.

## Сохранение

Демка использует browser `localStorage` key `kanban-aging-demo-state-v2-ru`. Backend отсутствует.
